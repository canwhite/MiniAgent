# 随机修正技巧库

## 什么是"随机扰动"？

在扩散模型中，随机扰动是让生成结果具有多样性的关键。在写作中，它是指：

> **在逻辑正确的前提下，加入一个意外的小细节，让文字产生"真人写作的意外感"。**

---

## 按场景分类的扰动示例

### 对话场景

**标准版（AI 味）：**
```
"我不同意，"他说，"这样做太危险了。"
```

**加入扰动：**
```
"我不同意，"他说，指尖无意识地敲击着桌面，"这样做太危险了。"
```
> 扰动元素：指尖敲击（暗示焦虑或思考）

---

### 动作场景

**标准版（AI 味）：**
```
他冲向敌人，挥剑攻击。
```

**加入扰动：**
```
他冲向敌人，脚下的碎石让他踉跄了一下，但顺势挥出了更强的一剑。
```
> 扰动元素：踉跄（增加真实感，顺势反击显示经验）

---

### 环境描写

**标准版（AI 味）：**
```
房间里很安静，只有钟表的滴答声。
```

**加入扰动：**
```
房间里很安静，只有钟表的滴答声——突然，一只苍蝇撞在了窗玻璃上。
```
> 扰动元素：苍蝇撞窗（打破单调，增加生活气息）

---

### 心理描写

**标准版（AI 味）：**
```
她感到很紧张，不知道该怎么办。
```

**加入扰动：**
```
她感到很紧张，不知道该怎么办，直到她注意到桌上那张泛黄的照片。
```
> 扰动元素：注意到照片（为后续情节埋伏笔，或触发回忆）

---

## 不同类型的扰动

### 1. 感官扰动
- **声音**：突然的噪音、远处的人声、自然的声音
- **视觉**：光影变化、移动物体、颜色对比
- **触觉**：温度变化、质感、疼痛
- **嗅觉/味觉**：突如其来的气味、味道

**示例：**
```
正说着，厨房传来焦糊的味道——大概是面包烤过头了。
```

### 2. 物品扰动
- **功能异常**：东西坏了、不工作了
- **意外出现**：不该出现的东西出现了
- **物品互动**：人物与物品的小互动

**示例：**
```
他想拿起杯子，却发现杯子底部和桌面粘在了一起。
```

### 3. 时间扰动
- **突然的停顿**：对话或动作的中断
- **时间跳跃感**：意识到时间过得很快/很慢
- **节奏变化**：快节奏中的慢瞬间，或反之

**示例：**
```
争吵还在继续，但他突然意识到，太阳已经完全落山了。
```

### 4. 关系扰动
- **意外的眼神**：读懂或误解了一个眼神
- **身体距离**：意外的靠近或疏远
- **微表情**：一闪而过的表情

**示例：**
```
她没有回答，但嘴角微微上扬了不到一秒钟。
```

## 使用原则

### ✅ 好的扰动

1. **不改变主线**：只是细节层面的意外
2. **增加真实感**：让场景更立体、更像现实
3. **有潜在意义**：可能暗示人物性格、或为后续埋伏笔
4. **不突兀**：虽然意外，但逻辑上合理

### ❌ 不好的扰动

1. **破坏逻辑**：为了让"意外"而让角色行为不合理
2. **喧宾夺主**：扰动变成了焦点，分散主线注意力
3. **纯粹炫技**：为了显得"不一样"而强行加元素
4. **前后矛盾**：与之前的设定冲突

---

## 如何生成好的扰动？

### 方法 1：观察生活
注意现实中那些"不完美"的瞬间：
- 说话时的小动作
- 环境中的微小变化
- 物品的随机行为

### 方法 2：反向思考
问自己：
- 这个场景"太完美"在哪？
- 真实情况下会出什么"小差错"？
- 什么细节会让读者感到"这很真实"？

### 方法 3：让 AI 帮忙
可以用这样的 prompt：
```
在这个场景中加入一个微小的、不改变剧情但增加真实感的细节。
要求：
1. 与人物当前的情绪或处境相关
2. 不是重大的情节转折
3. 让场景更有生活气息
```

## 示例对比

### 原文（标准 AI 生成）
```
李明走进办公室，同事们都忙碌地工作着。他来到自己的工位，
开始打开电脑。经理走过来，交给他一份文件。
"这个项目需要你完成，"经理说。
"好的，"李明回答。
```

### 加入扰动后
```
李明走进办公室，同事们忙碌的声音像潮水一样涌来。他来到
自己的工位，正要打开电脑，却发现鼠标垫不见了。

正疑惑着，经理走过来，把一份文件放在他桌上——刚好压在
了原本放鼠标垫的位置。

"这个项目需要你完成，"经理说，顺手理了理领带。

"好的，"李明回答，眼睛还在寻找着那个鼠标垫。
```

### 分析
1. **鼠标垫不见了** → 意外的小麻烦
2. **文件压在原位置** → 巧合但不影响剧情
3. **经理理领带** → 人物小动作
4. **李明还在找** → 延续的悬念线

这些扰动都没有改变主线（经理布置任务），但让场景更生动、更像真实发生的事。
